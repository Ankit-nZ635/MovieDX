# 🎬 MovieDX - Movie Discovery Platform

A full-stack MERN movie discovery platform where users can browse movies, watch trailers, see where to stream them, and explore upcoming releases.

## 🚀 Features
- Browse 50+ movies with real posters & IMDb ratings
- Watch trailers directly from YouTube
- Filter by genre and rating
- See where to watch (Netflix, Prime, Hotstar)
- Coming Soon section for upcoming movies
- User authentication (JWT + bcrypt)
- Admin panel to manage movies
- Email notifications via Nodemailer

## 🛠️ Tech Stack
- **Frontend:** React.js, Redux, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT + bcrypt
- **APIs:** OMDb API
- **Email:** Nodemailer

## ⚙️ Setup
git clone https://github.com/Ankit-nZ635/MovieDX.git
cd iCinema
npm install
cd frontend && npm install
cd ..
npm run dev
